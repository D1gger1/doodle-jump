🎮 Doodle Jump Game

A browser-based game inspired by Doodle Jump, built using JavaScript and HTML5 Canvas. The project focuses on game logic, physics simulation and real-time rendering.

🚀 Live Demo

👉 https://doodlejumpi.netlify.app/

🎯 Gameplay

Control the character and jump across platforms while avoiding falling down. The goal is to reach the highest score possible.

✨ Features
Game loop with continuous rendering
Gravity and jump physics simulation
Collision detection with platforms
Dynamic platform generation
Score tracking system
Smooth animations using JavaScript
🛠️ Tech Stack
JavaScript (Game Logic, Animation)
HTML5 Canvas
CSS
